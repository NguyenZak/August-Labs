# August Agency - System Architecture

## Overview
August Agency is a premium Marketing Agency Web App with a built-in Client Portal for tracking campaigns, reporting, and invoices. 

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Framer Motion
- **Database:** PostgreSQL
- **ORM:** Prisma
- **Authentication:** NextAuth.js
- **Icons:** Lucide React

## Folder Structure
- `/src/app` - Next.js App Router pages and layouts
- `/src/components` - Reusable UI components
- `/src/lib` - Utility functions (e.g., `cn` for Tailwind)
- `/src/services` - External API integrations or business logic
- `/src/types` - TypeScript interfaces and types
- `/prisma` - Database schema and migrations
- `/docs` - Project documentation
